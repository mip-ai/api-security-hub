Herramientas de seguridad API

Presentamos herramientas útiles para la evaluación, prueba y monitoreo de vulnerabilidades de API.

🔎 Herramientas de prueba dinámica de seguridad de aplicaciones (DAST)

Herramientas que envían solicitudes a API en ejecución para detectar vulnerabilidades.

Burp Suite Edición comercial/gratuita disponible

La herramienta estándar de la industria para pruebas de seguridad de API y aplicaciones web. Capacidades de interceptación y escaneo basadas en proxy.

  • Interceptar y modificar solicitudes/respuestas HTTP
  • Escaneo automatizado (inyección SQL, XSS, etc.)
  • Intrusa: Fuzzing de parámetros
  • Repetidor: repetición y verificación de solicitudes manuales
  • Ecosistema de extensión rico (BApp Store)
Consejos para las pruebas API

Importe una definición de OpenAPI/Swagger para rastrear y probar automáticamente todos los puntos finales.

OWASP ZAP Gratis / OSS

Una herramienta de prueba de seguridad de código abierto desarrollada por OWASP. Fácil de integrar en canalizaciones de CI/CD.

  • Escaneo activo/escaneo pasivo
  • Importar definiciones de API (OpenAPI, GraphQL)
  • Integración CI/CD (GitHub Actions, Jenkins)
  • Ejecución sin cabeza con soporte Docker
  • Pruebas personalizadas mediante scripting
BashEscaneo API con ZAP Docker
docker run -t ghcr.io/zaproxy/zaproxy:stable zap-api-scan.py \
  -t https://api.example.com/openapi.json \
  -f openapi \
  -r report.html

🧪 Herramientas de desarrollo y prueba de API

Cartero Gratis/Comercial

La herramienta de referencia para el desarrollo y las pruebas de API. También útil para pruebas de seguridad.

  • Organice casos de prueba con la función Colecciones
  • Cambiar entre puesta en escena/producción con variables de entorno
  • Adquirir tokens automáticamente con scripts de solicitud previa
  • Automatizar la validación de respuestas en la pestaña Pruebas
  • Ejecutar desde CI/CD con Newman (CLI)
JavaScriptEjemplo de guión de prueba de cartero
// Validación del código de estado
pm.test("Status is 200", () => {
  pm.response.to.have.status(200);
});

// Validación del encabezado de seguridad
pm.test("Security headers present", () => {
  pm.response.to.have.header("X-Content-Type-Options");
  pm.response.to.have.header("Strict-Transport-Security");
});

// Comprobar si hay exposición de datos confidenciales
pm.test("No sensitive data exposed", () => {
  const body = pm.response.json();
  pm.expect(body).to.not.have.property("password");
  pm.expect(body).to.not.have.property("password_hash");
  pm.expect(body).to.not.have.property("secret");
});

curl + jq Gratis/CLI

Pruebas API rápidas desde la línea de comando. Fácil de programar y automatizar.

  • Disponible en cualquier entorno
  • Automatizar con scripts de shell
  • Analizar respuestas JSON con jq
  • Inspección detallada del encabezado HTTP
BashComprobación de encabezado de seguridad
# Verificar encabezados de respuesta
curl -s -D - -o /dev/null https://api.example.com/health

# Intentar acceder sin autenticación
curl -s -w "%{http_code}" https://api.example.com/api/users

# Prueba BOLA (acceder al recurso de otro usuario)
curl -s -H "Authorization: Bearer $TOKEN" \
  https://api.example.com/api/users/OTHER_USER_ID

📊 Análisis estático (SAST) y análisis de composición de software (SCA)

Semgrep Gratis/OSS

Una herramienta de análisis estático basada en patrones. Fácil de crear reglas personalizadas.

  • Soporte en varios idiomas (JS, Python, Go, Java, etc.)
  • Conjuntos de reglas de seguridad integradas
  • Fácil integración CI/CD
  • Definir reglas personalizadas en YAML

Snyk Nivel gratuito disponible

Escaneo de vulnerabilidades de dependencia y análisis de código.

  • Escaneo de dependencias para npm, pip, Maven, etc.
  • Detección automática de CVE conocidos
  • Generar automáticamente relaciones públicas de corrección
  • Escaneo de imágenes de contenedores

auditoría npm / Trivy Gratis

Comprobación de vulnerabilidades para administradores de paquetes y contenedores.

  • npm audit: paquetes de Node.js
  • auditoría de pip: paquetes de Python
  • Trivy: contenedores, IaC y sistemas de archivos
  • Ideal para activación de CI/CD
BashComprobación de vulnerabilidad de dependencia
# Verificar las dependencias de Node.js
npm audit --production

# Escaneo de seguridad con Semgrep
semgrep --config=p/security-audit ./src

# Escaneo de contenedores con Trivy
trivy image --severity HIGH,CRITICAL myapp:latest

📝 Gestión de documentación y especificación de API

Swagger / OpenAPI Especificación estándar

Defina las especificaciones API en un formato legible por máquina. También se utiliza como entrada para pruebas de seguridad.

  • Definición integral de puntos finales, parámetros y respuestas.
  • Esquemas de seguridad explícitos (OAuth, clave API, etc.)
  • Importar a ZAP/Burp para automatizar las pruebas
  • Generación de documentación interactiva con Swagger UI

Espectral Gratis / OSS

Una herramienta linter para especificaciones OpenAPI. Comprueba el cumplimiento de las mejores prácticas de seguridad.

  • Verifique la presencia de definiciones de esquemas de seguridad
  • Detectar puntos finales no autenticados
  • Aplique políticas internas con reglas personalizadas
  • Puerta de calidad para especificaciones API en CI/CD

📡 Herramientas de monitoreo y registro

Puerta de enlace API

Kong, AWS API Gateway, Apigee y otras.

  • Autenticación y autorización centralizadas
  • Aplicación de limitación de tarifas
  • Solicitar agregación de registros
  • Alertas de detección de anomalías

WAF

AWS WAF, Cloudflare, ModSecurity y otras.

  • Bloquear patrones de ataque como SQLi/XSS
  • Control de acceso basado en GeoIP
  • Mitigación de bots
  • Definiciones de reglas personalizadas

SIEM/Análisis de registros

ELK Stack, Splunk, Datadog y otros.

  • Agregación y visualización de registros API
  • Detección de patrones anómalos
  • Soporte de respuesta a incidentes
  • Auditoría de cumplimiento
