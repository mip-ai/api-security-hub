Ferramentas de segurança de API

Apresentando ferramentas úteis para avaliação, teste e monitoramento de vulnerabilidades de API.

🔎 Ferramentas de teste dinâmico de segurança de aplicativos (DAST)

Ferramentas que enviam solicitações para APIs em execução para detectar vulnerabilidades.

Burp Suite Edição comercial/gratuita disponível

A ferramenta padrão do setor para testes de segurança de aplicações web e API. Capacidades de interceptação e varredura baseadas em proxy.

  • Interceptar e modificar solicitações/respostas HTTP
  • Verificação automatizada (SQL Injection, XSS, etc.)
  • Intruso: difusão de parâmetros
  • Repetidor: repetição e verificação manual de solicitação
  • Rico ecossistema de extensões (BApp Store)
Dicas de teste de API

Importe uma definição OpenAPI/Swagger para rastrear e testar automaticamente todos os endpoints.

OWASP ZAP Gratuito / OSS

Uma ferramenta de teste de segurança de código aberto desenvolvida pela OWASP. Fácil de integrar em pipelines de CI/CD.

  • Verificação ativa/verificação passiva
  • Importar definições de API (OpenAPI, GraphQL)
  • Integração CI/CD (GitHub Actions, Jenkins)
  • Execução sem cabeça com suporte Docker
  • Testes personalizados via script
BashVerificação de API com ZAP Docker
docker run -t ghcr.io/zaproxy/zaproxy:stable zap-api-scan.py \
  -t https://api.example.com/openapi.json \
  -f openapi \
  -r report.html

🧪 Ferramentas de desenvolvimento e teste de API

Carteiro Gratuito / Comercial

A ferramenta ideal para desenvolvimento e teste de API. Também útil para testes de segurança.

  • Organize casos de teste com o recurso Coleções
  • Alternar entre preparação/produção com variáveis ​​de ambiente
  • Adquirir tokens automaticamente com scripts de pré-solicitação
  • Automatize a validação de respostas na guia Testes
  • Execute a partir de CI/CD com Newman (CLI)
JavaScriptExemplo de script de teste do Postman
// Validação do código de status
pm.test("Status is 200", () => {
  pm.response.to.have.status(200);
});

// Validação do cabeçalho de segurança
pm.test("Security headers present", () => {
  pm.response.to.have.header("X-Content-Type-Options");
  pm.response.to.have.header("Strict-Transport-Security");
});

// Verifica a exposição de dados confidenciais
pm.test("No sensitive data exposed", () => {
  const body = pm.response.json();
  pm.expect(body).to.not.have.property("password");
  pm.expect(body).to.not.have.property("password_hash");
  pm.expect(body).to.not.have.property("secret");
});

curl + jq Grátis / CLI

Teste rápido de API na linha de comando. Fácil de criar scripts e automatizar.

  • Disponível em qualquer ambiente
  • Automatize com scripts de shell
  • Analise respostas JSON com jq
  • Inspeção detalhada do cabeçalho HTTP
BashVerificação de cabeçalho de segurança
# Verifique os cabeçalhos das respostas
curl -s -D - -o /dev/null https://api.example.com/health

# Tentativa de acesso sem autenticação
curl -s -w "%{http_code}" https://api.example.com/api/users

# Teste BOLA (acessando recurso de outro usuário)
curl -s -H "Authorization: Bearer $TOKEN" \
  https://api.example.com/api/users/OTHER_USER_ID

📊 Análise Estática (SAST) e Análise de Composição de Software (SCA)

Semgrep Gratuito / OSS

Uma ferramenta de análise estática baseada em padrões. Fácil de criar regras personalizadas.

  • Suporte multilíngue (JS, Python, Go, Java, etc.)
  • Conjuntos de regras de segurança integrados
  • Fácil integração CI/CD
  • Definir regras personalizadas em YAML

Snyk Nível gratuito disponível

Verificação de vulnerabilidades de dependência e análise de código.

  • Verificação de dependência para npm, pip, Maven, etc.
  • Detecção automática de CVEs conhecidos
  • Gerar PRs de correção automaticamente
  • Digitalização de imagens de contêiner

auditoria npm / Trivy Grátis

Verificação de vulnerabilidade para gerenciadores de pacotes e contêineres.

  • auditoria npm: pacotes Node.js
  • auditoria pip: pacotes Python
  • Curiosidades: contêineres, IaC e sistemas de arquivos
  • Ideal para gate CI/CD
BashVerificação de vulnerabilidade de dependência
# Verifique as dependências do Node.js.
npm audit --production

# Verificação de segurança com Semgrep
semgrep --config=p/security-audit ./src

# Verificação de contêiner com Trivy
trivy image --severity HIGH,CRITICAL myapp:latest

📝 Especificação de API e gerenciamento de documentação

Swagger/OpenAPI Especificação padrão

Defina as especificações da API em um formato legível por máquina. Também usado como entrada para testes de segurança.

  • Definição abrangente de endpoints, parâmetros e respostas
  • Esquemas de segurança explícitos (OAuth, chave de API, etc.)
  • Importe para ZAP/Burp para automatizar testes
  • Geração de documentação interativa com Swagger UI

Espectral Gratuito / OSS

Uma ferramenta linter para especificações OpenAPI. Verifica a conformidade com as melhores práticas de segurança.

  • Verifique a presença de definições de esquema de segurança
  • Detectar endpoints não autenticados
  • Aplique políticas internas com regras personalizadas
  • Porta de qualidade para especificações de API em CI/CD

📡 Ferramentas de monitoramento e registro

Gateway de API

Kong, AWS API Gateway, Apigee e outros.

  • Autenticação e autorização centralizadas
  • Aplicação de limitação de taxa
  • Solicitar agregação de log
  • Alertas de detecção de anomalias

WAF

AWS WAF, Cloudflare, ModSecurity e outros.

  • Bloqueie padrões de ataque como SQLi/XSS
  • Controle de acesso baseado em GeoIP
  • Mitigação de bots
  • Definições de regras personalizadas

SIEM/Análise de Log

ELK Stack, Splunk, Datadog e outros.

  • Agregação e visualização de logs de API
  • Detecção de padrão anômalo
  • Suporte à resposta a incidentes
  • Auditoria de conformidade
